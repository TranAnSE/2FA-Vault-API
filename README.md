# 2FA-Vault-API

Repository used to edit the 2FA-Vault API specification.

2FA-Vault is a web based self-hosted alternative to One Time Passcode (OTP) generators like Google Authenticator, designed for both mobile and desktop.

- 2FA-Vault API documentation: <https://docs.2fa-vault.app/resources/rapidoc.html>
- 2FA-Vault main repository: <https://github.com/TranAnSE/2FA-Vault>

## Validation

Install dependencies, then validate all OpenAPI specs:

```bash
npm install
npm run validate:openapi
```
